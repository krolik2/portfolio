My projects

Scan Plan

View

My first mobile app for android, made with React Native and Expo. The goal was to increase productivity of Gemini's drugstore warehouse employees, by enabling them to speedup container scanning process and save excessive movement. App provides list of qr codes and corresponding location names, which can be filtered and then assigned to a specific container. This helps user plan his walking path optimally.

Weather app

View

Weather app which uses Darksky weather and Aqi air quality apis, displayed data is based on user's browser geolocation. App shows also current time and date. Background image changes with each page refresh. This project was meant to practice async/await, api calls, callbacks and even though it's a rather small app I have applied BEM methodology to HTML/CSS.

TODO app

View

I've decided to learn React Hooks, Context api and Firebase Firestore, so I've made this app while learning. Each logged-in user, has access to his own list of todos, which are saved on firebase-firestore. Every item added by the user, can be edited, marked as completed and deleted.
